
# Matrix Playground using Python and NumPy

A beginner-friendly matrix operations project built using Python OOP and NumPy.

## Features

- Matrix Addition
- Matrix Subtraction
- Matrix Multiplication
- Matrix Division
- Matrix Transpose
- Determinant Calculation
- Row Sum
- Column Sum
- Maximum Element Detection

## Technologies Used

- Python
- NumPy

## How to Run

```bash
pip install numpy
python main.py
